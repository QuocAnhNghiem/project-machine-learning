Algorithm: Logistic Regression

1. Linear Score (z)

$$ z = \theta_0 + \theta_1 x_1 + \dots + \theta_n x_n = \theta^T x $$

Combines features (x) with learned weights (θ)

2. Sigmoid Activation (Probability)

$$ \hat{y} = \sigma(z) = \frac{1}{1 + e^{-z}} $$

Squashes the score z into probability range [0, 1]

3. Log Loss (Cost Function J)

$$ J(\theta) = -\frac{1}{m} \sum_{i=1}^m \left[ y^{(i)} \log(\hat{y}^{(i)}) + (1-y^{(i)}) \log(1-\hat{y}^{(i)}) \right] $$

Penalizes wrong predictions exponentially

4. Gradient Descent (Update)

$$ \theta_j := \theta_j - \alpha \frac{\partial J}{\partial \theta_j} $$

α = learning rate, iteratively minimizes cost

Decision Rule:   IF ŷ ≥ threshold → Predict Win (1)   |   ELSE → Predict Not Win (0)

L2 Regularization & Normalization

🛡️

L2 Regularization

Adds penalty for large weights to prevent model from memorizing noise (overfitting).

$$ J_{reg}(\theta) = J(\theta) + \frac{\lambda}{2m} \sum_{j=1}^n \theta_j^2 $$
  • Controls model complexity
  • λ (lambda) = regularization strength
  • Gradient updated: dw += (λ/m) · w
📏

Min-Max Normalization

Scales disparate features into the same standard range [0, 1].

$$ X_{norm} = \frac{X - X_{min}}{X_{max} - X_{min}} $$
  • Crucial: Parameters fit on Train set only!
  • Applied symmetrically to Test set
  • Ensures stable and fast Gradient Descent

Algorithm Implementation (model.py)

1. Sigmoid Function

@staticmethod
def _sigmoid(z):
    z = np.clip(z, -500, 500) # No overflow
    return 1.0 / (1.0 + np.exp(-z))

2. Log Loss Calculation

@staticmethod
def _log_loss(y_true, y_pred):
    eps = 1e-12
    p = np.clip(y_pred, eps, 1-eps) # No log(0)
    return -np.mean(
      y_true * np.log(p) + (1-y_true) * np.log(1-p)
    )

3. Gradient Descent Loop + L2

def fit(self, x, y):
    # Init weights
    self.w, self.b = np.zeros(x.shape[1]), 0.0

    for _ in range(self.epochs):
      # Forward probs
      p = self._sigmoid(np.dot(x, self.w) + self.b)

      # Raw gradients
      dw = (1/m) * np.dot(x.T, (p - y))
      db = (1/m) * np.sum(p - y)

      # Apply L2
      if self.l2 > 0: dw += (self.l2/m) * self.w

      # Update
      self.w -= self.lr * dw
      self.b -= self.lr * db
